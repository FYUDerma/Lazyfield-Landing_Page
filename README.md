## How to install

1. Clone this repository:

```bash
git clone https://github.com/FYUDerma/Lazyfield-Landing_Page.git
```

2. Go into project

```bash
cd Lazyfield-Landing_Page
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
